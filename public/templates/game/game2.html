Rappel code couleur

Les couleurs peuvent être représentées par différents codages. Un premier est le codage RGB qui à une couleur associe une nuance de rouge, une nuance de vert et une de bleu. Ces nuances sont des entiers pouvant aller de 0 à 255 en valeurs décimales. L'association de ces trois nuances permet la formation d'une couleur. Une autre représentation de ces couleurs peut être un entier hexadécimal composé de 6 chiffres.

Règles du jeu

Le but du jeu est de voir l'utilisation du binaire et de l'hexadécimal dans le codage des couleurs. Les règles sont les suivantes : A partir du codage décimal RGB de la couleur, il faut retrouver le codage hexadécimal. Un codage hexadécimal est constitué d'un # suivi de 6 chiffres hexadécimaux.
Exemple : #ABCDEF
Les deux premiers chiffres représentent la nuance de rouge allant jusqu'à 255 en décimal, les deux chiffres suivant la nuance de vert et les deux derniers chiffres la nuance de bleu.
Attention un chiffre héxadécimal X compris entre 0 et F s'écrira ici 0X (0 suivi du chiffre)
