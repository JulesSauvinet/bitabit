Règles du jeu

Le but du jeu est de décoder le message caché derrière les différents nombres codés en binaire. Il faut tout d'abord traduire le binaire en décimal, puis se référer à la table ASCII à gauche afin d'associer au nombre décimal la lettre correspondante. A chaque encadré noir représentant un mot codé en binaire, il faut associer une lettre de l'alphabet à rentrer dans le champ texte situé en dessous. Quand vous avez saisi TOUS les champs, vérifier votre réponse en cliquant sur le bouton de validation vert. Vous avez trois vies.

Rappel code ASCII

Le codage des caractères est une convention qui permet, à travers un codage connu de tous, de transmettre de l'information textuelle,là où aucun support ne permet l'écriture scripturale. Le code ASCII permet d'associer à chaque caractère un nombre.
